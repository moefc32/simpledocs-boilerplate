Simpledocs Boilerplate

Simpledocs is a minimalistic and efficient documentation site designed for designers, developers, and teams who need a quick and organized way to present their documentation. It offers a clean and responsive layout with essential features like a navigation sidebar and search functionality. Built with simplicity in mind, this project ensures that users can focus on writing content without dealing with complex configurations. Its lightweight nature makes it easy to deploy on the GitHub Pages.
